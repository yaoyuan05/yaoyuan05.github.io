移动端页面适配新思路

使用vw来实现页面的适配

遵守以下规范：

  • 对于设计稿的尺寸转换为vw单位，可使用Sass函数编译。或者用PostCSS的插件把px转换成vw
  • 无论是文本还是布局高宽、间距等都使用 vw 作为 CSS 单位

  • 1物理像素线（也就是普通屏幕下 1px ，高清屏幕下 0.5px 的情况）采用 transform 属性 scale 实现。

  • 为了更好的实现长宽比，特别是针对于img、vedio和iframe元素。实现思路padding-top: 百分比;推荐方法:

  • 为了解决视口过大或者过小，失去了最大最小宽度的限制，必要时给布局宽度最大和最小限制。
