jQuery源码学习笔记

jQuery.merge

可将类数组转为数组。jQuery.merge([],arrayLike);
可将数组转为类数组。jQuery.merge({“length”:”0”},array);
也可以用于合并数组。jQuery.merge(array,array);

jQuery.fn.css –> jQuery.access(elems, fn, key, value, chainable, emptyGet, raw);

jQuery.ajax

xhr.onreadystatechange = callback;
通过检查statechange改变来触发fn done;
回调函数是通过jQuery.Callbacks管理，当状态满足时触发回调

// Success/Error
if ( isSuccess ) {
    deferred.resolveWith( callbackContext, [ success, statusText, jqXHR ] );
} else {
    deferred.rejectWith( callbackContext, [ jqXHR, statusText, error ] );
}
